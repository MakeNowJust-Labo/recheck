Getting Started

JavaScript

Uses the following command to install this library:

$ npm install @makenowjust-labo/redos

This library exports only an API called check. It takes a RegExp pattern source and flags, and returns the analysis result.

$ node
> const { check } = require('@makenowjust-labo/redos');
undefined
> check('^(a|a)*$', '');
{
  complexity: 'exponential',
  witness: { pumps: [ [Object] ], suffix: '\x00' }
}

Scala

Adds the following line into your build.sbt:

libraryDependencies += "codes.quine.labo" %% "redos" % "0.1.0"

ReDoS object is a frontend of this library. You can use ReDoS.check method to analyze RegExp pattern.

$ sbt console
scala> import codes.quine.labo.redos.ReDoS
import codes.quine.labo.redos.ReDoS

scala> ReDoS.check("^(a|a)*$", "")
val res0: scala.util.Try[codes.quine.labo.redos.Checker.Complexity] = Success(Exponential(Witness(List((List([a]),List([a]))),List([\x00-`b-\uFFFF]))))
